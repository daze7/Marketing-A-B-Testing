# Marketing A/B Test Analysis

Pet-проект по продуктовому анализу A/B-теста рекламной кампании.

## Цель проекта

Проверить, увеличивает ли рекламная кампания конверсию пользователей по сравнению с контрольной группой `psa`.

В проекте анализируется A/B-тест на данных Kaggle **Marketing A/B Testing**.

Источник данных: https://www.kaggle.com/datasets/faviovaz/marketing-ab-testing

## Статистическая гипотеза

Обозначения:

- `CR_ad` — конверсия пользователей, которые видели рекламу;
- `CR_psa` — конверсия контрольной группы.

Гипотезы:

- **H0:** `CR_ad ≤ CR_psa`, рекламная кампания не увеличивает конверсию;
- **H1:** `CR_ad > CR_psa`, рекламная кампания увеличивает конверсию.

Тестовая статистика рассчитывается при граничном случае H0: `CR_ad = CR_psa`.

## Параметры эксперимента

- **Основная метрика:** Conversion Rate
- **Statistical method:** one-sided two-sample z-test for proportions
- **Significance level:** α = 0.05
- **MDE:** 0.5 percentage points

## Данные

В проекте используется датасет **Marketing A/B Testing**.

Основные поля:

| Поле | Описание |
|---|---|
| `user_id` | идентификатор пользователя |
| `test_group` | группа эксперимента: `ad` или `psa` |
| `converted` | факт конверсии |
| `total_ads` | количество просмотренных объявлений |
| `most_ads_day` | день, когда пользователь увидел больше всего объявлений |
| `most_ads_hour` | час, когда пользователь увидел больше всего объявлений |

## Метрики

### Conversion Rate

```text
Conversion Rate = Converted Users / Total Users
```

### Absolute uplift

```text
Absolute uplift = CR_ad - CR_psa
```

### Relative uplift

```text
Relative uplift = (CR_ad - CR_psa) / CR_psa
```

## Что сделано

- загружены и проверены данные;
- проверены пропуски, дубликаты и распределение пользователей по группам;
- рассчитаны conversion rate, absolute uplift и relative uplift;
- проведён one-sided z-test для сравнения двух долей;
- построен 95% доверительный интервал для разницы конверсий;
- проверено достижение MDE;
- проведён χ²-тест как дополнительная проверка связи между группой и конверсией;
- выполнен сегментный анализ по дням недели, часам и количеству показов;
- сформулирован продуктовый вывод и ограничения анализа.

## Основные результаты

По результатам анализа:

- конверсия в группе `ad` выше, чем в группе `psa`;
- рост конверсии статистически значим на уровне α = 0.05;
- доверительный интервал для разницы конверсий полностью находится выше нуля;
- при MDE = 0.5 п.п. эффект можно считать продуктово значимым, если нижняя граница доверительного интервала превышает MDE.

## Итоговый вывод

Рекламная кампания статистически значимо повышает конверсию пользователей по сравнению с контрольной группой `psa`.

Кампанию можно рекомендовать к масштабированию при условии, что дополнительная выручка от роста конверсии покрывает стоимость рекламы.
